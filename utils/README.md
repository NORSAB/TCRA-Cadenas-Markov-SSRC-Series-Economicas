# TCROC-Adicionales — Scripts Auxiliares

Scripts para generar figuras, tablas de apéndices y snippets LaTeX de la tesis.

## Contenido

| Script | Genera |
|--------|--------|
| `generar_allserie_NNLS.py` | Figura comparativa NNLS todas las series |
| `generar_apendice_datos_reales.py` | Tablas LaTeX del Apéndice A3 |
| `generar_apendice_figuras_reales.py` | Figuras del Apéndice (grafos TikZ) |
| `generar_evolucion_probabilidades.py` | Evolución temporal de probabilidades |
| `generar_grafico_complejidad.py` | Gráfico de complejidad computacional |
| `generar_grafico_pib_proyecciones.py` | Proyecciones PIB Honduras |
| `ejemplo_combustibles_metricas.py` | Métricas de ejemplo para combustibles |
| `update_latex.py` | Actualización de snippets LaTeX |
